The question

The map registers node labels drawn from a fixed doc-type vocabulary. Whether that vocabulary should be closed to the fourteen named types or open to arbitrary strings has never been settled — it was assumed closed and never tested.

The kernel already carries kernel_landmark, which no Rust schema names, and the cogmap birth path stamps cogmap_charter. So the closed reading is already false in production; the only question left is whether that was intended.
